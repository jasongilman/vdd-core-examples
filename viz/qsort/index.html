About

This is for visualizing code implementing the Quick Sort algorithm. The visualization is implemented using D3.

The code being visualized is located in vdd-core-examples/src/sorter/. The visualization code is in vdd-core-examples/viz/qsort/.

Driving in the REPL

The quick sort code is run every time the submit button is pressed. The output is captured and sent to the visualization. The quick sort code can also be tested and visualized from the REPL. The following code can be run to view quick sorting a shuffled list of 20 numbers.

(require '[qsort.driver :as q])
(q/viz-qsort (shuffle (range 20)))

Note: This page must be open to visualize the code execution.
